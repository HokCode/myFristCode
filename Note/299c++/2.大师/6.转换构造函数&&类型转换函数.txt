41.类型转换函数1
	
	(1)c语言类型转换
		
		1.小类型数据转向大类型数据支持隐式的类型转换。
	
		2.两个不同类型的数据相加，后者会转换为前者的类型。
	
		例子：
			short s;
			sizeof(s + 'b') = 4;
		
		3.编译器为了高效为把s和'b'隐式转换为int类型，所以大小为4。
	
	(2)转换构造函数
		
		1.转换构造函数可将其他类型转换为类类型。
		
		2.只有一个参数，参数类型是基本类型或其他类类型。
			
			Test (int i)
			{
				mValue = i;
			}
		
			Test t;
			t = 100;
			t + 10;
		
		3.此时编译器会到Test类中寻找转换构造函数，通过生成临时对象进行赋值。相当于：
		
			t = Test(100);t + Test(10)。
	
		4.编译器会尽力尝试隐式类型转换，有可能导致工程bug。
	
	(3)explicit关键字
		
		1.用explicit关键字杜绝编译器的转换尝试。
		
		2.用explicit修饰的转换函数只能显示转换，如：
			
			static_cast<ClassName>(value);
	
42.类型转换函数2
	
	(1)类类型转换为其他类型
		
		1.使用类型转换函数。语法规则：
			
			operator Type ()
			{}
		
		2.与转换构造函数类似，编译器也会尝试寻找类型转换函数。如：
			
			Test t;
			int i = t;
			等价于：int i = t.operator int()。
	
	(2)类与类之间的类型转换
		
		1.可使用转换构造函数或类型转换函数
	
		2.为了避免二义性，转换构造函数都会有explicit需要事故，让编译器只会自动调用类型转换函数。
	
		3.工程中类型转换函数的隐式调用会导致bug，所以会用公有成员函数代替类型转换函数，格式为：
			
			Type toType();
